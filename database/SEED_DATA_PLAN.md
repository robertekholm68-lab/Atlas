# Seed Data Plan

## Purpose

Define the initial data required for the MVP prototype and first implementation.

## Required Seed Data

- Muscle taxonomy
- Exercise catalogue
- Exercise activation profiles
- Recovery status labels
- Design colour tokens for body states

## Exercise Data Rule

MVP uses an in-house authored seed set.

Do not copy proprietary exercise datasets.

Exercise names may be common terms, but descriptions and activation profiles must be authored specifically for ATLAS.

## Minimum Exercise Categories

- Horizontal push
- Vertical push
- Horizontal pull
- Vertical pull
- Squat
- Hinge
- Lunge
- Core
- Arms
- Calves

## Licensing Posture

All seed data should be original or clearly license-compatible.

If external reference material is used, it must not be copied verbatim.
