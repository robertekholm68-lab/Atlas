# UX Blueprint

MVP focus: design the product experience before production code.

Primary flow: open Mission Control, inspect the body, review next action, log a session, complete session, see body state update.

Core screens: login, onboarding, Mission Control, Body Map, Muscle Card, Session Logging, Exercise Library, Profile baseline settings.

Body interaction: zoom, pan, controlled rotation, front/back switching, selectable regions and persistent spatial context.
