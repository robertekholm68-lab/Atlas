# Success Metrics

Version: 1.0
Status: MVP Foundation

## Purpose

This document turns ATLAS product goals into measurable outcomes.

## MVP Metrics

1. Time to first meaningful action on Mission Control should be under 10 seconds.
2. A user should be able to complete the first logged session in under 3 minutes after selecting exercises.
3. Muscle state should update immediately after a completed session.
4. At least one recommendation explanation should be visible without needing to open a separate AI chat.
5. The Body Map should make the current recovery state understandable without reading a chart.

## Review Cadence

Metrics should be reviewed after every prototype test and before each MVP milestone.
