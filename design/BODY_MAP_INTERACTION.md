# Body Map Interaction

## Purpose

Define how the user interacts with the ATLAS body interface.

## MVP Interaction Model

- Click or tap a visible region to select it.
- Selected region opens a detail panel.
- User can switch between front and back views.
- User can zoom in and out.
- User can pan the body view.
- User experiences controlled rotation between front and back.

## 3D Direction

The product vision includes a fully spatial body.

MVP should feel spatial through 2.5D interaction.

Full free-rotation 3D is deferred until the core loop is validated.

## Accessibility

Every selectable region needs:

- Accessible name
- Keyboard focus
- Text state
- Numeric state where relevant

A list alternative must exist for users who cannot use the visual map.
